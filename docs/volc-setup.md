# 火山引擎 ARK 配置指南

## 1. 注册火山引擎账号
- 访问 https://www.volcengine.com/
- 完成实名认证

## 2. 开通大模型方舟（ARK）
- 进入控制台 → 机器学习平台 → 大模型方舟
- 开通服务（有免费额度）

## 3. 创建 API Key
- 大模型方舟 → API Key 管理 → 创建 API Key
- 复制 Key（格式：Bearer 后面的字符串）

## 4. 获取模型接入点
- 进入「在线推理」→ 创建推理接入点
- 选择模型：**Doubao-1.5-vision-pro-250328**（视觉模型）
- 创建后会得到 endpoint ID

## 5. 配置环境变量
在 Vercel 环境变量中添加：
```
ARK_API_KEY=your-api-key-here
```

## 6. 重新部署
修改环境变量后需要重新部署才能生效。

## P图程度检测原理
通过技术指标估算：
- **文件大小**：过小（多次压缩）或异常都有问题
- **熵值分析**：过度平滑（磨皮）会降低熵值
- 最终得分 0-10，越高P得越狠

## 费用参考

- 一张普通照片约 1000-3000 token
- 单次评分成本约 0.003-0.01元
